rentuacar.com 2021

Сайт аренды автомобилей.

Rentuacar предоставляет огромный выбор автомобилей на прокат, начиная от эконом класса, вроде Nissan Micra, заканчивая премиальными моделями электромобилей по типу Tesla и внедорожников — Audi Q7 и Toyota Land Cruiser 200, которыми заказчик может управлять самостоятельно без водителя.

О проекте.

Основная задача - разработать современный и удобный сайт для аренды автомобилей, внедрить фильтрацию для удобного подбора авто на прокат для пользователей сайта, реализовать технические seo-требования для успешного продвижения проекта в поисковых системах.

Каталог Rentuacar - это сотни автомобилей, при этом прайс на каждый автомобиль устанавливается в зависимости от количества дней аренды и типа аренды (с залогом или без). При этом, на сайте важно в доступном и не перегруженном для пользователя виде показать весь каталог авто с подробной информацией, реализовать удобное оформление заказа на аренду авто, выбор дополнительных опций автомобиля.
